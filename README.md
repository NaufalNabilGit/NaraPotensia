# NaraPotensia

Pastikan kamu sudah mendownload repositori ini.
Ekstrak file yang sudah didownload.
Letakkan repositori di dalam folder htdocts xampp kamu.
Buka aplikasi xampp dan klik start pada apache dan mysql.
cp .env.example .env
File env dan sesuaikan dengan sistem kita
Buka CMD dan arahkan ke direktori ini.
Ketik "composer install" dan tekan enter
ketik "php artisan key:generate" dan tekan enter
ketik "php artisan migrate" dan tekan enter
ketik "php artisan db:seed" dan tekan enter
ketik "php artisan serve" dan tekan enter
Buka web browser dan tulis 'http://127.0.0.1:8000/' di form url lalu tekan enter.
